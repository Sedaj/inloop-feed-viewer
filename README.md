# Inloop Feed Viewer {frontend}

Features:

 * Webpack ( dev/prod config )
 * single build file
 * Typescript
 * SaSS 7-1 + webpack loader + autoprefixer
 * Hot-Reload module
 * Redux
 * Persistent after page reload (localStorage)

 > Installing

```
npm install
```

 > Running in development mode 

```
npm run dev
```

 > Building
 
```
npm run build
```
